# Employee Salary Data Transformation

## Introduction

This project shows how to transform employee salary data using Python, Pandas, and NumPy. It includes creating new columns, working with dates, categorizing salaries, and finding outliers.

## Tools Used

* Python
* Pandas
* NumPy
* VS Code / Jupyter Notebook

## Tasks Performed

* Created an employee dataset.
* Calculated the total salary by adding salary and bonus.
* Converted joining dates into datetime format.
* Extracted the joining year from the joining date.
* Categorized employees based on their salary.
* Detected salary outliers using the IQR method.

## Conclusion

This project helps understand basic data transformation techniques used in data preprocessing. These methods make the data easier to analyze and prepare for future data science tasks.
