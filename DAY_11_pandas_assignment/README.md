# Student Performance Analysis Using Pandas

## Introduction
This project analyzes student performance data using the Pandas library in Python. It helps understand students' marks and attendance through different data analysis techniques.

## Tools Used
- Python
- Pandas
- Jupyter Notebook / VS Code
- CSV File

## Dataset
The dataset contains the following columns:
- Student_ID
- Name
- Department
- Gender
- Marks
- Attendance

## Tasks Performed
- Read the CSV file using Pandas.
- Group data by department and calculate statistics.
- Rank students based on marks within each department.
- Filter departments based on average marks.
- Merge student data with scholarship data.
- Create pivot tables for marks and attendance.
- Export reports to CSV files.

## Conclusion
This project helped in learning important Pandas concepts such as GroupBy, Aggregation, Filter, Merge, Pivot Tables, and exporting data. These techniques are useful for performing real-world data analysis.
