# Wine Quality Data Analysis using Python

## Project Overview

This project demonstrates basic Exploratory Data Analysis (EDA) using Python, Pandas, Matplotlib, and Seaborn on a small Wine Quality dataset.

The goal is to inspect the dataset, understand data distributions, identify relationships between variables, and visualize important patterns.

---

## Dataset

The dataset contains the following features:

| Column | Description |
|----------|-------------|
| quality | Wine quality score |
| alcohol | Alcohol percentage |
| sugar | Sugar content |

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install required libraries

```bash
pip install pandas matplotlib seaborn
```

3. Open Jupyter Notebook

```bash
jupyter notebook
```

4. Run all notebook cells.
