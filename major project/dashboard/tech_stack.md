# Technology Stack

## Core platform
- Python 3.11 for data generation, cleaning, and automation
- Streamlit for the interactive dashboard experience
- Pandas and NumPy for data preparation and transformation

## Visualization and analytics
- Altair for polished interactive charts
- Matplotlib and Seaborn for supporting analysis plots
- SciPy for statistical testing and deeper insight generation

## Data and storage
- CSV-based processed tables for the fact and dimension layers
- SQL queries stored in the database folder for structured analysis
- A clear fact-and-dimension design for analytical reporting

## Collaboration and delivery
- GitHub for version control and project sharing
- VS Code as the main development workspace
- A Vercel-friendly static website structure for presentation and documentation
